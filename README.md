
[WIP] Woodenlist
------

A realtime Todolist.

### Workflow

To run the client side app:

```bash
boot build-advanced
npm i
webpack
```

Files will be compiled at `target/`. Open `index.html`.

To start the server:

```bash
boot build-simple
npm i
cd target/
node main.js
```

For more details, read the docs of the Workflow:

https://github.com/Cumulo/cumulo-workflow
